# Google Sites setup checklist (optional / low priority)

## Why this matters
Google Sites can provide a lightweight verifiable presence that backs up the Google Business Profile. Don't duplicate the main site; use it as a quick "yes this business exists" page.

## Steps

### 1. Create the site
- [ ] Go to https://sites.google.com
- [ ] New site → blank template
- [ ] URL: sites.google.com/view/wabash-systems (or similar)

### 2. Content (single page is enough)
- [ ] Logo at top
- [ ] One-line tagline: "E-commerce consulting for SMBs nationwide — Midwest-rooted."
- [ ] Brief description (2-3 sentences from the public site's about section, rewritten short)
- [ ] CTA: "Visit wabashsystems.com →" linking to the main site
- [ ] Contact: email link
- [ ] Address: Robinson, IL (matching GBP)

### 3. Publish + link
- [ ] Publish
- [ ] Add the URL to the Google Business Profile as the business website (NO — point GBP at wabashsystems.com instead; the Google Sites page is supplementary)
- [ ] Add the URL to the schema on index.html as an additional `sameAs` entry

### 4. Don't overthink it
- This is a 30-minute task. Don't build it out. The main site is wabashsystems.com.
